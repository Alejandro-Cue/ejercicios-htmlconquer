LA ETIQUETA <code>.

Volver atrás.

Se usa para indicarle al navegador que su contenido es código fuente (de cualquier lenguaje). Es una etiqueta de línea y se renderiza con la fuente predeterminada monospace que tenga el navegador y tiene atributos comunes y de evento. Lo normal sería darle estilos css asociados al tipo de lenguaje.

Ejemplo de uso de la etiqueta <code>.

En python, si quiero impimir una variable llamada name tendré que ejecutar el siguiente código: print(name) y dicha variable será impresa en pantalla.
