LA ETIQUETA <blockquote>.

Volver atrás.

La etiqueta <blockquote> se utiliza para indicar que el texto pertenece a un sitio externo del cual se indica la fuente.

Es una etiqueta de bloque que tiene todo los atributos comunes y de eventos y tiene un atributo específico, cite, de tipo url que indica el origen de la cita.

Ejemplo de etiqueta <blockquote>.

La web lenguajehtml.com nos advierte de algunos malos usos de esta etiqueta.

"Una etiqueta que no suele ser bien comprendida en el Lenguaje HTML es la etiqueta <blockquote>. Muchos desarrolladores utilizan las etiquetas de enlaces <a> o la de párrafos <p> para escribir un texto y enlazar la fuente, sin embargo, HTML tiene herramientas mucho más apropiadas para este objetivo."
