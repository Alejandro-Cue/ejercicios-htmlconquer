Ejemplo de textarea.

Atributos:

  • Required: hace obligarorio rellenar el campo para poder enviar el formulario. Se coloca dentro de Select.
  • Selected: coloca por defecto uno de los campos del Select. Se coloca dentro del Option que quieras poner por defecto.
  • Múltiple: permite seleccionar varios campos a la vez tiene que ir acompañado de atributo Size. Se colocan ambos atributos en el select. Size haría referencia al número de campos que se van a ver simultaneamente una vez se renderice.
